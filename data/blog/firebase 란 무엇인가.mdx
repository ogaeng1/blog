## Firebase(파이어베이스) ?

간단히 말해 운영체제에 상관없이 앱을 만들수 있게 도와주는 개발 툴이라고 한다.
구글 드라이브와 애널리틱스를 적용해 어떤 기기에서도 개발할 수 있는 환경을 제공하며, 이용자들의
방문 횟수나 광고 효과, 문제 발생 빈도를 알려줘 쉽게 개발할 수 있도록 지원해 준다고 한다.

## Firebase 몇 가지 기능을 알아보자

> 인증

인증 기능은 사용자가 로그인 및 회원가입을 할 때, 보다 쉽고 편리하게 할 수 있도록 도와주는 기능이라고 한다.
페이스북이나 트위터, Github SDK를 사용할 필요 없이 파이어베이스의 인증 기능을 이용한 SSO(통합 인증)을
지원하는 서비스를 통해 사용자가 로그인 활동을 쉽게 할 수 있다고 한다.

**SSO(Single Sign On)** <br />
단 한번의 인증 절차만으로도 여러 앱에 접속 가능한 인증 방식을 말한다.

<br />

> 실시간 데이터베이스

파이어베이스는 Nosql클라우드 데이터베이스에 JSON형태로 데이터를 저장하고 클라이언트에 실시간으로 동기화 한다.

<br />

> 클라우드 파이어스토어

파이어베이스 및 구글 클라우드 플랫폼의 웹, 모바일 서버 개발에 사용되는 유연하고 확장이 가능한 DB.

<br />

> 스토리지

파이어베이스의 저장소는 사용자가 제작한 이미지, 오디오, 동영상 등 컨텐츠들을 저장 합니다.

<br />

> 호스팅

파이어베이스는 HTML, CSS, JS 등과 같은 정적 컨텐츠를 빠르고 안전하게 호스팅 해준다. 또한 SSL을
제공해 주기 때문에 안전하게 컨텐츠를 전송할 수 있으며, 전세계 CDN에 캐싱되어 있기 때문에
사용자 위치에 대한 제약없이 컨텐츠를 빠르게 전송할 수 있다.

**호스팅** <br />
인터넷상의 공간을 임대해주는 서비스로 직접 웹 서버를 구축하지 않아도 서버 일부 공간을 임대해서
독자적인 서버를 가진것과 동일한 효과를 갖는다.
